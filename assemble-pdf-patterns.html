Assemble PDF Patterns with PDFStitcher for Inkscape



When beginning in sewing, it is useful to know about how to work with your pdf pattern to turn it into an A0 layout especially for projector sewing to make any alteration, lengthen/shorten or change the pattern in some way.

But first, make sure that you have downloaded and installed the required software below.

Download PDFStitcher
Download Inkscape


PDFStitcher: Generate A0 File


Open up PDFStitcher. In the options tab, first select the pdf pattern you want to work with. Then where you want to save the pdf pattern. The next required field is the page range you want to assemble together.




In the tile pages tab, for required paramters add the number of columns or number of rows. Change the page order or page rotation if needed. For optional parameters, you could add margins to the a0 pdf pattern and if your chosen pattern has any white margins around each page then you could add values to trim it from each side. For the final file, press the Generate PDF button. Under the Layers tab, you could see any layers for sizes, labels or grid.
