Create a cube with Inkscape




1. Go to create stars and polygons, select regular polygon and choose 6 corners, change the color and stroke of the object if you like. I made mine transparent in grey.




2. Right-click on the object and duplicate the object three times.



3. Place on the corners like so.




4. Select division from Path in the menu.




5. This splits the polygon object enabling you change colors and work with the parts of it as a cube.
